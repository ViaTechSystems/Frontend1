Directions

The goal is to turn the select options into a well styled drag-and-drop interface. Any number of "Not Assigned Options" should be able to be selected and then dragged over to the "Assigned Options" and vice-versa (dragging works both ways). Once options are dragged over, they should continue to be selected in the event the user wishes to transfer them back.

The height of the boxes holding the options should be responsive and fill the entire height of the remaining visible screen. Meaning if the screen size increases, the boxes grow. If the screen size decreases, the boxes shrink to fit.

If you have any question ask Matt. You can change whatever code you want in this document.
